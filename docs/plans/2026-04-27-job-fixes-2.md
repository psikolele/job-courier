# Job Courier Translation and UI Polish Implementation Plan

> **For Antigravity:** REQUIRED WORKFLOW: Use `.agent/workflows/execute-plan.md` to execute this plan in single-flow mode.

**Goal:** Centralize translations using JSON files, fix responsive text overflow in the Navbar mega menu, and extract the AdBanner component to ensure 50/50 graphical consistency across the application.

**Architecture:** We will set up `i18next` JSON resources (`en.json`, `it.json`, etc.) to replace hardcoded strings. The mega menu will use Tailwind classes (`break-words`, `whitespace-normal`) for responsive wrapping. A new `AdBanner.jsx` component will unify the advertisement design across `Filters.jsx` and `Home.jsx`.

**Tech Stack:** React, Tailwind CSS, react-i18next

---

### Task 1: Setup i18next JSON configuration

**Files:**
- Modify: `src/i18n.js` (or create it if not present, and import in `main.jsx` / `App.jsx`)
- Create: `src/locales/it.json`, `src/locales/en.json`, `src/locales/de.json`, `src/locales/fr.json`

**Step 1: Write the minimal implementation**
We will create the JSON files with keys for the Navbar and Hero, and configure `i18n.js` to load them.

*(We will provide translations for it, de, fr as well).*

**Step 2: Commit**
