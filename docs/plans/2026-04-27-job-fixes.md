# Job Courier Fixes Implementation Plan

> **For Antigravity:** REQUIRED WORKFLOW: Use `.agent/workflows/execute-plan.md` to execute this plan in single-flow mode.

**Goal:** Fix translations, responsive padding for menus, enable touch swiping for latest offers, correct text truncation, and restore the backend job fetching service.

**Architecture:** We will modify `Navbar.jsx` for i18n logic and CSS classes, adjust `Filters.jsx` to replace marquee with standard horizontal scrolling (snap-x), update the Vercel API `api/jobs.js` to fetch from the specified jobroom URL, and add a cron configuration to `vercel.json` to regularly hit the API.

**Tech Stack:** React 19, Tailwind CSS, Vercel Serverless, Cheerio.
